# 6일차 학습

## 조건 처리
* 조건문 : 지정한 조건이 `true`인 경우 실행하는 명령의 집합

### `if ~ else` 문
* `if` : 특정 조건이 참인 경우 실행
* `else` : `if`의 조건이 거짓일 경우 실행

### 거짓으로 판단하는 값
* `false`, `undefined`, `null`, `0`, `NaN`, `''`(공백)

## 연산자 × 조건 처리

### 비교 연산자
+ `===` : 엄격한 같은
+ `!==` : 엄격한 다른 
+ `>`   : - 보다 큰
+ `>=`  : - 보다 크거나 같음
+ `<`   : - 보다 작음
+ `<=`  : - 보다 작거나 같음

* 엄격한 규칙을 적용하는 것을 권장

### 논리 연산자
* `A && B` : A, B 모두 값이 참이여야만 `true`
* `A || B` : A, B 둘 중 하나의 값이 참이면 `true`
