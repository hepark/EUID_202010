# 4일차 학습

## 동일한 변수 이름 문제
* js에서 html요소를 선택할 때 `id` 값을 사용하지 않는다. 
* 다른 사용자에 의해 `id` 값과 동일한 이름의 변수를 지정하게 되면 먼저 선택한 `id` 값의 유형이 변경될 수 있다. (스크립트의 `hoist`기능으로 인해!)

## 문서객체에 접근하는 방법
* 이전에 문서객체에 접근하는 방법
  + `getElementBy`기능을 사용하여 문서에 접근했다.

* 최근에 문서객체에 접근하는 방법
  + `.querySelector`, `.querySelectorAll`을 사용해서 문서 객체에 접근한다.
