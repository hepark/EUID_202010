# Git CLI 커맨드 모음 
sourcetree나 github for Desktop같은 git GUI보다는 CLI (command-line interface) 커맨드로 git 관리를 하는 데에 익숙해져야 한다고 해서 자료를 찾고 써먹는 대로 일단 다 기록해두는 git CLI 커맨드 방앗간🦥 <br />
CLI를 써야 git의 모든 기능을 다 제어할 수 있다고 하던데...하면서 알게 되겠지...🤖<br />
